# Expense Tracker (CLI)

A simple command-line expense tracker written in Python.
It allows users to record daily expenses, view them by date, and save the data to a text file.

## Features

- Add expenses with amount and reason
- View expenses grouped by date
- Save expenses to a file
- Load existing expenses from file
- Simple CLI menu system

## File Format

Expenses are stored in `data.txt` using this format:

date|amount|reason

Example:

12-12-2025|120|food
12-12-2025|50|bus

## How to Run

1. Make sure Python is installed
2. Run the script:

python main.py
